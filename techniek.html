Technieken

HTML

Voor de structuur van de website heb ik verschillende HTML-elementen gebruikt:

  • <div>: Gebruikt om secties van de pagina te groeperen.
  • <header>: Bevat de navigatiebalk en de titel van de pagina.
  • <ul> en <li>: Gebruikt voor de navigatielijst.
  • <footer>: Bevat de footer van de pagina.
  • <section>: Gebruikt om inhoudelijke secties te groeperen.

CSS

Voor de opmaak van de website heb ik de volgende CSS-technieken gebruikt:

  • Kleuren en lettertypes: Ik heb gekozen voor een minimalistisch ontwerp met een donkere achtergrond en witte tekst. Het lettertype "Raleway" wordt gebruikt voor alle tekst.
  • Layouttechnieken: Ik heb CSS Grid gebruikt om de inhoud van de pagina's netjes te organiseren. De header en footer zijn vastgezet aan de boven- en onderkant van de pagina.
  • Media queries: Ik heb media queries gebruikt om de website responsief te maken, zodat deze goed werkt op verschillende schermformaten.

JavaScript

Voor de interactieve elementen van de website heb ik de volgende JavaScript-functionaliteiten toegevoegd:

  • Event listeners: Ik heb een event listener toegevoegd aan de knop op de homepage om extra content te tonen of te verbergen wanneer erop wordt geklikt.
  • Dynamische content: De knop op de homepage verandert van tekst afhankelijk van de zichtbaarheid van de extra content.
